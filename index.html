QU’EST-CE QUE LA VEILLE TECHNOLOGIQUE ?


La veille technologique est une activité qui consiste à se tenir au courant des dernières avancées technologiques dans un domaine.
La veille technologique dans le domaine de l’informatique est utilisée principalement pour :

- Suivre les évolutions techniques,
- Dégager de nouveaux procédés,
- Identifier les meilleures pratiques.


MES PRINCIPAUX OUTILS DE VEILLE TECHNOLOGIQUE


Pour enrichir mes connaissances dans le domaine de l'informatique, je m'efforce de maintenir une veille technologique active.
